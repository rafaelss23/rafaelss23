### 📂 Projetos em Destaque

#### 🤖 [Esteira Automatizada de Monitoramento de Vagas e Alertas](https://github.com/rafaelss23)
* **Descrição:** Solução híbrida desenvolvida para eliminar o monitoramento manual de oportunidades em portais de vagas.
* **Tecnologias:** Python (Selenium, Pandas) + n8n / Power Automate.
* **Como funciona:** Um robô em Python realiza o *web scraping* dos dados brutos diretamente da web, o Pandas realiza o tratamento dos dados estruturando-os, e a plataforma de automação orquestra as regras de negócio, filtrando as palavras-chave mais relevantes e enviando alertas automáticos formatados por e-mail ou canais de comunicação.

#### 📊 [Dashboard de Eficiência Operacional e KPIs](https://github.com/rafaelss23)
* **Descrição:** Projeto de Business Intelligence focado em monitorar o desempenho de processos automatizados, ajudando na identificação de gargalos operacionais.
* **Tecnologias:** SQL Server, Power BI (DAX e Power Query).
